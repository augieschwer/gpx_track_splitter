# gpx_track_splitter
Split a GPX file with multiple tracks into separate GPX files

```
positional arguments:
  input_file  Path to the input GPX file

options:
  -h, --help  show this help message and exit
```

```
python3 gpx_track_splitter.py Current.gpx
```
